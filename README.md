# QuickQuiz

1. Interactive Web Quiz Application:
    A dynamic, fully responsive web application that challenges users with randomized questions across multiple subjects. The app tracks performance in real-time and provides a detailed analytical dashboard upon completion.

2. Key Features:
    ⏱ Dynamic Timer: Automatic submission when time runs out.
       
    📊 Results Dashboard: Displays marks, accuracy, Correact , Incorrect and Percentage  using localStorage.
   
    🎲 Randomized Questions: Picks 10 random questions from a JSON database.
   
    📱 Responsive Design: Optimized for mobile, tablet, and desktop views.

4. Tech Stack
    Frontend: HTML5, CSS3 (Grid & Flexbox), JavaScript (ES6+).
   
    Data: JSON (stored locally in Assets/).
   
    Storage: Browser LocalStorage API.

6. About Files:
    The project mainly contains 3 pages the homepage(index.html) , quizpage(quiz_page.html) , dashboardpage(dashboard.html). with a Assest folder which contain the Json file for questions

7. Procedure to run the project:
    1. Use any code editor.
    2. Clone the repo from github.
    3. Run the index.html in web browser by hosting a local server.
